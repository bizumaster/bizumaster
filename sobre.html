Objetivo

O sistema foi criado para auxiliar militares no cálculo de soldo, adicionais, descontos e indenizações, sempre em conformidade com as legislações vigentes.

Equipe

Este projeto é fruto de estudo em programação com foco em JavaScript, HTML e CSS, usando arquivos JSON para organização dos dados.
